# NeonBounceLab - 霓虹弹力球实验室

## 概述
一个纯前端赛博朋克风格的霓虹弹力球交互动画实验。50个发光小球在 Canvas 上自由弹跳，带有拖尾残影、碰撞检测、连线效果和鼠标交互。

## 技术栈
- HTML5 Canvas
- CSS3 (自定义属性 + 动画)
- Vanilla JavaScript (ES6+)
- 无后端、无依赖

## 目录结构
- \CODE/index.html\ — 主页面（HTML + CSS + JS 单文件）
- \README.md\ — 项目说明
- \PROJECT_PLAN.md\ — 项目计划

## 快速开始
直接用浏览器打开 \CODE/index.html\ 即可运行。
