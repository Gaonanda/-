# NeonBounceLab - 项目计划

## 目标
创建一个本地运行的霓虹弹力球交互式动画网页，展示赛博朋克风格的视觉效果。

## 功能清单
- [x] 全屏 Canvas 动画背景
- [x] 50 个霓虹发光小球初始生成
- [x] 小球随机运动 + 边界反弹
- [x] 小球之间的弹性碰撞
- [x] 运动拖尾残影效果
- [x] 近距离发光连线
- [x] 鼠标吸引效果
- [x] 点击生成 10 个新球
- [x] 左上角实时数据显示
- [x] 右下角 3 个控制按钮

## 里程碑
1. 项目脚手架搭建 - 2026-07-21 ✅
2. Canvas 动画核心逻辑 - 2026-07-21 ✅
3. 碰撞检测与连线 - 2026-07-21 ✅

5. UI 样式完善 - 2026-07-21 ✅

## 风险评估
- Canvas 性能：小球过多可能掉帧 → 已做空间划分优化
- 碰撞精度：弹性碰撞计算复杂 → 使用简化动量守恒
